# README

## 介绍

本工具基于wails开发的一个简单的端口转发工具，可以添加、删除、查询PC上的端口转发规则
如果你已经厌烦了不断的重复使用冗长的netsh命令来管理PC上的端口转发，那你可以使用本工具来简单的管理PC上的端口转发规则
